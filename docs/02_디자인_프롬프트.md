# FishBook 디자인 생성 프롬프트 (클로드 붙여넣기용)

> 아래 `===` 사이 내용을 통째로 복사해서 클로드(아티팩트)에 붙여넣으면 돼.
> 결과가 마음에 안 들면 맨 아래 "수정 요청 예시"를 이어서 쓰면 됨.

===

너는 시니어 프로덕트 디자이너야. "FishBook"이라는 **회 도감 웹 서비스**의 UI를 디자인해줘. 인터랙티브하게 클릭되는 **단일 파일 React 아티팩트(React + Tailwind, 외부 이미지 없이 플레이스홀더 사용)** 로 만들어줘.

## 제품 개요
- 한 줄 정의: 회에 관심 있는 사람이 "내가 먹는/먹을 회가 어떤 생선인지, 언제가 제철인지, 맛은 어떤지"를 둘러보고 검색하는 회 도감.
- 타겟: 회 입문자 · 마니아 · 횟집 손님 · 수산시장 구경객
- 핵심 행동: 둘러보기 + 검색하기 + 후기 읽고 쓰기
- 플랫폼: **웹 우선(반응형)**. 데스크탑/모바일 둘 다 자연스럽게.

## 디자인 방향
- 톤: **깔끔·미니멀**. 여백 넉넉하게, 정보는 한눈에.
- 메인 컬러: **바다 Teal**. 포인트 컬러로만 절제해서 사용(버튼, 활성 필터, 제철 뱃지). 배경은 화이트/연한 뉴트럴.
- 그라데이션·진한 그림자·요란한 효과 금지. 얇은 보더(0.5~1px) + 부드러운 라운드 카드.
- 사진이 주인공: 생선 사진을 큼직하고 선명하게.

## 디자인 토큰 (이대로 적용)
- Primary(Teal): `#0F9488` / hover `#0B7C72` / 연한 배경 틴트 `#E6F4F2`
- 텍스트: 본문 `#1A1D1F`, 보조 `#6B7280`, 흐린 `#9AA0A6`
- 배경: 페이지 `#FFFFFF`, 카드 `#FFFFFF`, 섹션 구분 `#F7F9FA`
- 보더: `#E8EBED`
- 별점/강조 포인트(선택): 앰버 `#F2A93B`
- 폰트: 한글 **Pretendard**(없으면 system sans-serif), 제목 weight 600, 본문 400
- 라운드: 카드 16px, 버튼/인풋 10px, 칩 999px(pill)
- 가격대 표기: ₩ / ₩₩ / ₩₩₩ (저렴/중간/고급)

## 만들 화면 4개 (한 앱 안에서 클릭으로 이동되게)

### 1) 둘러보기(목록) — 홈
- 상단: 로고(FishBook) + 네비(도감 / 제철 캘린더 / 후기)
- 검색창: "생선 이름 검색 (예: 광어, 방어)"
- 필터: 제철(봄/여름/가을/겨울) 칩, 맛(담백/기름진/쫄깃/고소) 칩 — 선택 시 Teal 활성
- 생선 카드 그리드(반응형): 사진, 이름, 제철 뱃지, 한 줄 소개, 맛 태그, 가격대(₩), 별점·후기수
- 카드 클릭 → 상세 페이지로 이동

### 2) 생선 상세
- 큰 대표 사진(+썸네일 여러 장 자리)
- 이름(국문/영문), 제철 뱃지, 가격대, 평균 별점
- 맛 특징(태그 + 설명 문장)
- 제철 정보(월/계절 시각화 — 예: 12개월 바 또는 캘린더)
- "비슷한 생선" 가로 카드 리스트
- 하단: 후기 섹션(목록 + "후기 쓰기" 버튼)

### 3) 검색 / 필터 결과
- 검색어 입력 상태 + 필터(제철·맛·가격대) 사이드/상단
- 결과 개수 표시, 정렬(인기순/이름순)
- 결과 없을 때 빈 상태(empty state) 디자인 포함

### 4) 후기(리뷰)
- 특정 생선의 후기 목록: 닉네임, 별점, 내용, 작성일, (선택)사진
- "후기 쓰기" 폼(모달 또는 인라인): 닉네임, 별점(1~5), 내용, 사진 첨부, 삭제용 비밀번호
- 익명 기반(로그인 없음)

## 더미 데이터
광어, 방어, 우럭, 참돔, 연어, 도미 등 6~8종을 그럴듯한 한국어 정보(제철·맛·가격·한줄소개)로 채워서 화면이 비어 보이지 않게 해줘.

## 산출물 요구
- 단일 파일 React 컴포넌트, Tailwind 클래스만 사용, 외부 라이브러리 최소화
- 4개 화면을 상단 탭 또는 라우팅 상태로 전환, 카드 클릭으로 상세 진입
- 모바일 반응형 포함
- 접근성: 적절한 대비, 버튼/인풋 레이블

먼저 전체를 한 번에 구현하고, 그다음 내가 부분별로 수정 요청할게.

===

## 수정 요청 예시 (이어서 쓰기)
- "상세 페이지의 제철 정보를 12개월 막대 그래프로 더 직관적으로 바꿔줘."
- "카드가 너무 빽빽해. 여백을 늘리고 사진 비율을 4:3으로."
- "Teal이 좀 강해. 포인트는 버튼이랑 제철 뱃지에만 쓰고 나머진 뉴트럴로."
- "후기 작성 폼을 모달로 분리해줘."
